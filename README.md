# E90ClusterOLED  
A simple OLED screen for the e90 BMW cluster  

Hardware:  
ESP32  
SSD1306 0.96 OLED 128x64  
12v to 5v step down  
Can Transceiver for ESP32  
E90 BMW gauge cluster  

Software Packages:  
Arduino Studio  
Adafruit SSD1306 Library https://github.com/nhatuan84/esp32-can-protocol-demo
